# Arch installer
## What is this? 
A simple script that one can download, and run within the Arch Linux install iso to install it automatically, including: 
- Picking between different Linux kernels (Regular, LTS, Zen, and Hardened).
- Choosing an environment, currently just XFCE (Base and Full), KDE (Base and Full), GNOME (Minimal, Base, And Extra), and i3.
- Optionally installing OpenSSH, FastFetch and/or TMux automatically. 
- Automatically enabling os-prober on GRUB on install.
- Automatically setting a default editor from `nano`, `vim`, `emacs`, or
